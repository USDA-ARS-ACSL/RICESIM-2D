Repository is for RICESIM-2D, a new rice model developed by USDA-ARS-ACSL integrated with 2DSOIL.

Current version executes but has numerous bugs and source code knowledge gaps that need to be addressed.  See documents folder for information.

Includes ports of multiple components of the ORYZA rice model modified by Sanai Li.  Most components have been/are being updated and modified to reflect model improvements, US production conditions and new experimental responses
